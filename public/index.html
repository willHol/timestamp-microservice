Timestamp Microservice

A microservice which parses and transforms UNIX or Natural Language dates

The API can be provided either a natural language date in the form: December 7, 1998

Alternatively, you may chose to provide the date as a UNIX timestamp: 912988800

Requests may be made as follows:

https://timestamp-ms.herokuapp.com/December%207,%201998

https://timestamp-ms.herokuapp.com/912988800

The API will return the following JSON response in both cases:

{"unix":344995200, "natural":"December 7, 1980"}
